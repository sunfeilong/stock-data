## 项目说明

收集上市公司列表和各个公司每分钟的股票价格。目前只收集国内股票数据。

每天16:30定时执行数据收集任务。收集完成之后自动推送数据到该仓库。

**数据:**

数据保存在[./storage](./storage)目录。

数据文件命名:

* 公司数据: `company-年-月-日.json` 
* 股票数据: `data-年-月-日.json`
